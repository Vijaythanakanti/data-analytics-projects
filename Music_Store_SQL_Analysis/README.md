# Music Store Data Analysis (SQL)

## Objective
Analyze music store sales data to understand customer behavior, top-selling artists,
popular genres, and revenue trends using SQL.

## Tools Used
- MySQL
- SQL (Joins, Subqueries, Aggregations, Window Functions)

## Key Business Questions
- Which artists and genres generate the most revenue?
- Who are the top customers by spending?
- How does revenue vary by genre and region?

## Approach
- Cleaned and validated the dataset
- Used joins and aggregations to answer business questions
- Optimized query performance using indexes

## Key Insights
- Identified top-performing artists and genres by revenue
- Found high-value customers contributing to majority of sales
- Improved query performance by ~30% using indexing
