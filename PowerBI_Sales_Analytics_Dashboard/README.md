# Sales Analytics Dashboard (Power BI)

## Objective
To analyze sales performance and provide actionable business insights using interactive
Power BI dashboards.

## Tools Used
- Power BI
- Data Modeling
- DAX (Basic Measures)
- Data Cleaning

## Key KPIs
- Total Sales
- Total Orders
- Profit
- Region-wise Performance
- Category and Sub-category Analysis
- Monthly Sales Trends

## Dashboard Features
- Interactive visuals with filters and slicers
- Trend analysis to identify growth patterns
- Regional and category-level performance insights

## Business Insights
- Identified top-performing regions and product categories
- Highlighted sales trends across time periods
- Enabled quick decision-making through KPI visibility
